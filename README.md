### Computer Vision & Deep Learning

**DDRNet23-Slim: Real-Time Semantic Segmentation**  
*PyTorch, CUDA, Distributed Training*

Research on efficient semantic segmentation architectures for real-time applications in autonomous systems and medical imaging.

- Achieved 63.92% mIoU on CamVid dataset at 102 FPS
- Implemented deep supervision and multi-resolution feature fusion
- Optimized training pipeline with Automatic Mixed Precision (AMP) and Distributed Data Parallel (DDP)
- Potential applications in real-time medical image segmentation

**Explainable AI: Backdoor Attack Detection with Grad-CAM**  
*PyTorch, ResNet50, Adversarial Machine Learning*

Research project investigating adversarial vulnerabilities in deep neural networks with focus on medical imaging safety.

- Simulated backdoor attacks and adversarial triggers on ResNet50
- Applied Grad-CAM for visual explanation of model decision-making
- Analyzed differences between clean and poisoned model inference patterns
- Contributed to understanding of AI trustworthiness and security in healthcare
